# ASDE: Agent Skill Development Engine

This repository contains a collection of bioinformatics Agent Skills built for the **Microsoft Agent Framework**. 

## Current Skills
- **pacsomatic**: Somatic variant calling for PacBio HiFi data.
- **SNV/Indel Caller**: (In Development) Specialized callers by Haidong and Wenchao.

## Usage
These skills are designed to be discovered and executed by AI Agents compatible with the Microsoft Agent Skills specification.
